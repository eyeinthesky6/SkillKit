# Overview

SkillKit is a framework for running repository automation skills in a secure sandbox.

- Sandboxed runtime (path globs, command allowlist, no shell injection)
- Audits and stats for every run
- CLI and JSON-RPC for automation and IDE integrations

See also: [Getting Started](getting-started.md), [Skills](skills.md), [Security](security.md), [Audits](audits.md).
