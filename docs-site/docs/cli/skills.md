# Skills Commands

## tsk skill:load

Load a skill (terminal-aware, cross-platform).

```bash
tsk skill:load <skill-name>
```

### Examples

```bash
# Load PDF skill
tsk skill:load pdf

# Load Excel skill
tsk skill:load excel

# Load database skill
tsk skill:load database
```

## tsk skills:add

Install a skill from GitHub.

```bash
tsk skills:add <user/repo>
```

### Example

```bash
tsk skills:add trinity-os/example-skill
```

## tsk list

List all installed skills.

```bash
tsk list
```

## tsk manage

Manage installed skills (update, remove, etc.).

```bash
tsk manage
```
